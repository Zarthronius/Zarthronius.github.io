Group Project: 3D jewelry mould e-commerce website

Involves: React (JS), STRIPE API Integration

Github repo

This website is a functioning prototype, that allows users to view and purchase 3d printable Jewelry moulds. This website was primarily developed in React. This site features a functioning Strapi database and content management system (CMS), in addition to Stripe payment integration.

This project was developed with the guidance and feedback of an external client who would ultimately be capable of hosting the website for day to day use.
